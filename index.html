IKANOW API Code Samples

The IKANOW API Code Samples project is a set of simple code examples built using HTML, JavaScript, and JQuery (with some additional help from Twitter's Bootstrap and CodeMirror) to demonstrate how to connect to and use the IKANOW Developer API. Important: Before interacting with the API you will need to sign up for a free API Key at: http://developer.ikanow.com/member/register.

The IKANOW API Code Samples project repository is available on GitHub: https://github.com/cvitter/ikanow.api.code.samples.

Code Samples

Person

api/social/person/get/: The Person Get API returns the Person JSON object for the User ID passed in (or for the current user if no ID is passed). The User Account Information sample page shows how to fetch user information and parse it using JQuery.

    Public Communities

    api/community/getpublic/: The Get Public Communities API returns a list of all communities that are publicly visible (whether or not the user calling the API is a member of each community). The sample code shows how to retrieve the list and simple ways of interacting with the community JSON object.

    Query Generator

    The Query Generator is an interactive tool useful for learning how to write query objects for the Document Query API (as well as demonstrating simple techniques for developing applications that interact with the IKANOW Developer API using JavaScript and JSP).
